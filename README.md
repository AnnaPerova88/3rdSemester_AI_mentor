# 3rdSemester_AI_mentor

# AI-mentor - проект, на основе RAG и LLM (Perplexity.API), который мы делали в 3 семестре магистратуры МИФИ. Задача заключалась в том, чтобы создать чат-бот, способный давать рекоммендации по программе обучения в области AI и быть репетитором, для освоения любых тем, связанных с ML, DL, CV, NLP.
